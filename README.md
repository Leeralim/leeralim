### 👨‍💻 Обо мне

Привет! Меня зовут Валерия. Я инженер-программист баз данных с опытом работы в научно-исследовательской сфере. Каждый божий день работаю с PostgreSQL: проектирую и оптимизирую запросы, создаю хранимые процедуры и инструменты для наших учёных, которые анализируют данные океанографических экспедиций, тралово-акустических съёмок и других исследований. Особое внимание уделяю целостности и сохранности данных: в наших базах хранятся уникальные записи с 1800-х годов, включая рейсы первого отечественного научно-исследовательского судна «Персей».❤️

* Создаю системы для обработки, загрузки и выгрузки научных данных, автоматизации расчётов и подготовки отчётности, что позволяет исследователям работать с данными быстрее и точнее, использовать их для обмена с нашими норвежскими коллегами из биологического института (Институт морских исследований (БИМИ) в Бергене).

* Люблю разбираться в структуре баз данных, анализировать планы выполнения запросов и искать способы оптимизации. Проще говоря, рыться под капотом СУБД. Честно говоря, тот процесс приносит мне удовольствия больше, чем программирование на питоне.
